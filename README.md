
# 🏰 Tile Palace

A simple 2D Unity platformer where the player collects coins, avoids enemies and hazards, and progresses through multiple levels.

Built as a practice project to get hands-on with Unity fundamentals — including prefabs, multi-scene management, simple animations, tilemaps, and a persistent game session system.

Hopefully the first of many. Will revisit Unity and build more games when time allows! 🎮

---

## 🛠️ Built With

- Unity 2D
- C#
- Tilemaps & Prefabs
- Unity Input System
- TextMeshPro

---

## 🕹️ Controls

| Action | Key |
|--------|-----|
| Move | A / D or Arrow Keys |
| Jump | Space |
| Climb | W / S near ladders |
| Shoot | Left Click |

---

## 📌 What I Learned

- Scene management and level transitions
- Persistent game state across scenes (DontDestroyOnLoad)
- 2D physics, colliders, and tilemaps
- Prefabs and reusable game objects
- Basic animations using Unity Animator
- Building and deploying a Unity WebGL project
